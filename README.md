# SafePay Mobile

A premium, investor-ready Expo / React Native starter app for **SafePay** — a wearable payment control app for smart rings and smartwatches.

## Included flows
- Splash and onboarding
- Sign up, login, OTP verification
- Home dashboard
- Add / verify / manage cards
- Pair devices and view device details
- Wearable control center
- Transaction history and receipt
- Notifications
- Security center
- Settings
- Help & support
- Profile

## Stack
- Expo
- React Native
- Expo Router
- TypeScript
- Linear gradients and custom design system

## Start locally
```bash
npm install
npm run start
```

Then open in:
- Expo Go
- iOS simulator
- Android emulator

## Important note
This is a **mobile app project**, so it is intended for Expo / native mobile preview, not Vercel deployment.
